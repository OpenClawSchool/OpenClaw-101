# 🤖 Sub-Agent Creation Flow

새로운 서브에이전트를 만들 때의 전체 프로세스와 자동화 가이드를 정리한 문서입니다.

## 1. 🚀 생성 요청 (Request)

아래 템플릿을 채워서 전달해 주세요.

```markdown
1. **이름 / 호출 라벨**
   - 예: Atlas, Neo, 클로드 등

2. **역할·책임 요약**
   - 어떤 업무를 맡기는지 한 줄 요약

3. **주요 Responsibilities (3~4개)**
   - 정기적으로 수행할 태스크 리스트

4. **Reference 자료 / 초기 파일**
   - `knowledge/` 또는 `reports/`에 기본으로 넣을 문서가 있다면 경로/파일명

5. **선호 모델/툴**
   - 기본으로 사용할 모델, 필수 도구, 사용 제한 등
```

## 2. 🛠️ 자동화 스크립트 (Auto-Generation)

아래 스크립트를 실행하면 자동으로 `AGENTS.md` 항목 추가, `agents/<name>/` 생성, INDEX 업데이트까지 처리됩니다.

```bash
#!/bin/bash

# 1. 입력값 받기
read -p "Agent Name: " AGENT_NAME
read -p "Role Summary: " ROLE_SUMMARY
read -p "Responsibilities (comma separated): " RESPONSIBILITIES
read -p "Reference File Path (optional): " REF_PATH
read -p "Preferred Model (optional): " MODEL

# 2. 디렉토리 생성
mkdir -p "agents/$AGENT_NAME"

# 3. AGENTS.md 업데이트
cat << EOF >> AGENTS.md

### $AGENT_NAME
- **Role**: $ROLE_SUMMARY
- **Responsibilities**: $RESPONSIBILITIES
- **Model**: ${MODEL:-gpt-4o}
- **Tools**: [default]
EOF

# 4. INDEX.md 업데이트
cat << EOF >> INDEX.md

- **$AGENT_NAME**: $ROLE_SUMMARY
EOF

# 5. Reference 파일 복사 (있을 경우)
if [ -n "$REF_PATH" ]; then
  cp "$REF_PATH" "agents/$AGENT_NAME/"
fi

# 6. 완료 메시지
echo "✅ Agent $AGENT_NAME created successfully!"
```

## 3. 📝 수동 생성 가이드 (Manual)

자동화 스크립트 대신 수동으로 만들 경우 아래 순서를 따르세요.

### Step 1: 디렉토리 생성
```bash
mkdir -p agents/<agent-name>
```

### Step 2: AGENTS.md 업데이트
```markdown
### <agent-name>
- **Role**: <한 줄 요약>
- **Responsibilities**: <주요 업무 3~4개>
- **Model**: <모델명>
- **Tools**: <사용할 도구>
```

### Step 3: INDEX.md 업데이트
```markdown
- **<agent-name>**: <한 줄 요약>
```

### Step 4: Reference 파일 추가 (선택)
```bash
cp <source> agents/<agent-name>/<filename>
```

## 4. 🎯 예시 (Example)

### 요청
```markdown
1. **이름 / 호출 라벨**
   - Atlas

2. **역할·책임 요약**
   - 주간 업무 보고서 자동 생성 및 데이터 분석

3. **주요 Responsibilities**
   - 매주 금요일 오후 5시에 전주 업무 데이터 취합
   - 주요 성과 및 이슈 분석
   - 보고서 초안 작성 및 GitHub에 푸시

4. **Reference 자료**
   - knowledge/weekly-report-template.md

5. **선호 모델/툴**
   - Claude 3.5 Sonnet
   - Tools: browser, file_system
```

### 생성 결과
- `agents/atlas/` 디렉토리 생성
- `AGENTS.md`에 Atlas 항목 추가
- `INDEX.md`에 Atlas 항목 추가
- `agents/atlas/weekly-report-template.md` 복사
