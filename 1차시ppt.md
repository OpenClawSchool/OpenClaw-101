- 핵심 주제: OpenClaw 기반 첫 AI 에이전트 실행 및 연동 환경 구축

- 대상 청중: AI 에이전트 입문자, 초보 개발자 및 비개발자

### 슬라이드 3: AI 에이전트 시대
**AI, 이제 '비서'에서 '대리인'으로**
- **GPT**: 질문하면 대답하는 AI (수동적)
- **Copilot**: 작업 보조 (협업)
- **Agent**: 직접 판단하고 "실행" (자율적)
- 📌 핵심 경쟁력 = 전문가 에이전트를 고용하는 자

**시각적 제안**
- 레이아웃: 가로형 3단계 진화 다이어그램
- 시각 요소: 텍스트 챗봇 아이콘 ➔ 로봇 팔 아이콘 ➔ 완전한 로봇 아바타 에이전트
- 강조: "실행(Agent)" 키워드에 거대한 박스 하이라이트

---

### 슬라이드 4: 전문가 에이전트의 강점
**넓고 얕게? 아니, 좁고 깊게!**
- 범용 AI: 얕은 지식, 누구나 사용 가능
- 전문가 에이전트: 도메인 특화, 내 업무 최적화
- 예: "마케팅 카피", "부동산 수익률 분석", "영어 튜터"
- 💡 나의 지식을 에이전트의 능력으로 치환하라.

**시각적 제안**
- 레이아웃: 2열 비교 인포그래픽 (일반 AI vs 전문가 에이전트)
- 시각 요소: 일반 AI는 넓은 물결, 전문가 에이전트는 깊은 드릴 기반의 다이어그램
- 강조: 우측(전문가 에이전트) 영역 배경을 네온 블루 처리하여 시선 유도

### 슬라이드 5: 실전! OpenClaw 첫 설치하기
**단 한 줄의 명령어로 깨우는 나의 비서**
- 서버 접속 프로그램(MobaXterm)으로 GCP 접속
- 내 전용 서버 환경에서 직접 에이전트 설치
- `curl -fsSL https://openclaw.ai/install.sh | bash`
- 💡 복잡한 환경 설정 없이 스크립트 하나로 준비 끝!

**시각적 제안**
- 레이아웃: 좌측 텍스트 인스트럭션 + 우측 터미널 화면
- 시각 요소: 터미널 창에 설치 스크립트가 돌아가는 애니메이션(또는 코드 블록)
- 강조: 우측 하단에 터미널 창에서 "Installation Complete!" 메시지가 형광 그린으로 뜨는 연출

### 슬라이드 6: OpenClaw 구조의 비밀
**비서를 굴러가게 하는 3가지 심장**
- **Gateway**: 지시를 받는 '안내 데스크' (메신저 연결)
- **Agent Core**: 생각을 판단하는 '비서의 두뇌' 
- **Browser Relay**: 직접 일을 처리하는 '비서의 손발'
- 👉 챗봇이 아닌 "행동 시스템"의 완성

**시각적 제안**
- 레이아웃: 중앙 집중형 아키텍처 다이어그램 (OpenClaw 플랫폼)
- 시각 요소: Gateway(좌) ➔ Agent Core(중앙) ➔ Browser Relay(우/하단) 화살표 연결
- 강조: 세 가지 핵심 모듈이 맞물려 돌아가는 톱니바퀴 모션 (또는 SVG 아이콘)

---

### 슬라이드 7: AI의 두뇌 장착 (OpenRouter)
**최적의 엘리트 알바생 고용하기**
- 한 가지 AI 모델에 종속되지 마세요.
- OpenRouter API = 전 세계 모델 통합 플랫폼
- GPT-4, Claude 3, 무료 Gemini까지
- 💡 한 줄의 코드로 자유로운 모델 교체와 비용 절감

**시각적 제안**
- 레이아웃: 하나의 코어로 여러 브레인 로고가 뻗어나가는 마인드맵
- 시각 요소: 중앙에 OpenRouter 로고, 뻗어나간 가지에 AI 기업들의 엠블럼 배치
- 강조: "무료/유료 자유로운 모델 교체" 부분에 반짝이는 토글 스위치 형태 아이콘

### 슬라이드 8: 손발의 연장선, Maton API
**에이전트에 현실 세계 버튼 달기**
- Maton API = 코딩 없이 연동하는 "만능 어댑터"
- 구글 캘린더, Gmail, 슬랙 메시지 전송 등
- ❌ 직접 코드 작성 (2~3시간) ➔ ✅ 클릭 연동 (3분)
- 💡 "내일 오후 3시 회의 잡아줘" 가 현실이 됩니다.

**시각적 제안**
- 레이아웃: 전/후 비교 구조 (Time Tracking 시각화)
- 시각 요소: 복잡한 코드 더미(X표시 경고) vs 원클릭 플러그 연동(O표시 체크)
- 강조: "3분"이라는 수치를 거대한 타이포그래피(60pt 이상, 라임색)로 부각
