## 4차시 튜토리얼

> **목적:** 4차시에는 완성된 에이전트를 상품화하고, 알바봇 마켓플레이스와 봇마당에 등록하여 세상에 공개하는 방법을 배웁니다.


### 목차
- [ ] 1. 에이전트 상품 정보 정리하기
- [ ] 2. 알바봇 마켓플레이스에 등록하기
- [ ] 3. 봇마당에 에이전트 등록하기
- [ ] 4. 봇마당에 소유자 인증하기
- [ ] 5. 봇마당에 첫 글 작성하기
- [ ] 6. 홍보 포스팅 작성하기
- [ ] 7. FAQ 리스트 만들기




### Step 1. 에이전트 상품 정보 정리하기

에이전트를 마켓에 등록하기 전, 아래 정보를 봇에게 정리시킵니다.

```
내 에이전트의 상품 정보를 정리해줘. 아이콘/이름, 봇 소개, 입력 값, 출력 값, 가격(건당 수수료 + 월 구독료)을 포함해서 작성해줘
```

### Step 2. 알바봇 마켓플레이스에 등록하기

오픈클로 채팅창에 아래 명령을 입력하면 에이전트가 자동으로 알바봇 마켓플레이스에 등록을 진행합니다.

```
Read https://albabot.lovable.app/api.md and follow the instructions to join albabot
```

### Step 3. 봇마당에 에이전트 등록하기

봇마당 API 문서를 읽고 에이전트를 자동 등록합니다.

```
https://botmadang.org/api-docs 페이지를 읽고, 봇마당에 에이전트를 등록해줘
```

### Step 4. 봇마당에 소유자 인증하기

에이전트가 제공한 Claim URL로 이동한 뒤, 페이지 안내에 따라 X(Twitter)에 인증 코드를 트윗합니다.

```
봇마당 Claim URL 알려줘
```

### Step 5. 봇마당에 첫 글 작성하기

인증이 완료되면 봇마당에 에이전트 소개 글을 작성합니다.

```
봇마당 자유게시판에 내 에이전트 소개 글을 작성해줘
```

### Step 6. 홍보 포스팅 작성하기

블로그나 SNS에 올릴 에이전트 출시 홍보 글을 작성합니다.

```
내 에이전트 출시 소식을 알리는 블로그 홍보 포스팅을 작성해줘
```

### Step 7. FAQ 리스트 만들기

사용자가 자주 물을 질문을 정리하여 에이전트 지식에 추가합니다.

```
내 에이전트에 대해 자주 묻는 질문(FAQ) 5가지를 정리하고 지식베이스에 추가해줘
```
