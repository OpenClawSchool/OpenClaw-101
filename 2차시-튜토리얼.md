## 2차시 튜토리얼

> **목적:** 2차시에는 에이전트의 핵심 지식 파일을 이해하고, Obsidian을 활용하여 전문가 Knowledge Base(KB)를 구축하는 방법을 배웁니다.


### 목차
- [ ] 1. 옵시디언 설치하기
- [ ] 2. 옵시디언 CLI 설치하기
- [ ] 3. 옵시디언 볼트 설정하기
- [ ] 4. PARA 기법 폴더 만들기
- [ ] 5. 핵심 노트 작성하기
- [ ] 6. 지식 연결하기
- [ ] 7. Knowledge 리스트 정리하기




### Step 1. 옵시디언 설치하기
```
옵시디언 설치해줘
```

### Step 2. 옵시디언 CLI 설치하기
```
옵시디언 CLI 설치해줘
```

### Step 3. 옵시디언 볼트 설정하기
```
현재 워크스페이스를 옵시디언 기본 볼트 폴더로 설정해줘
```

### Step 4. PARA 기법 폴더 만들기
```
옵시디언 파라 기법 폴더 만들어줘
```

### Step 5. 핵심 노트 작성하기

아래 포맷을 참고하여 내 전문 분야의 핵심 노트 5개를 작성합니다.

```
내 전문 분야에 대해 AI 친화적 문서 구조(Topic-Concept-Example-Checklist)로 핵심 노트 5개 작성해줘
```

### Step 6. 지식 연결하기

작성한 노트들을 `[[링크]]`로 연결하여 지식망을 구축합니다.

```
작성한 노트들을 옵시디언 링크로 서로 연결해줘
```

### Step 7. Knowledge 리스트 정리하기

에이전트가 답변할 수 있는 지식 범위를 정리합니다.

```
내 에이전트의 Knowledge Scope 문서를 정리해줘
```
